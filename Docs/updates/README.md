## üìã System Summary

### What is Edu-Pro?

A comprehensive school management platform supporting:

- **5 User Roles:** Admin/Principal, Teacher, Student, Coach, Librarian
- **8 Core Modules:** Authentication, School Management, Academic Structure, Attendance, Examination, Library, Sports, Communication
- **15 Database Collections:** Complete data model for all operations
- **100+ API Endpoints:** RESTful API with JWT authentication

### Technology Stack

| Layer    | Technology                                    |
| -------- | --------------------------------------------- |
| Frontend | React 19.1.0, Vite 7.0.0, Tailwind CSS 3.4.17 |
| Backend  | Node.js 22.16+, Express 4.21.2                |
| Database | MongoDB 8.0+, Mongoose 8.9.4                  |
| Auth     | JWT 9.0.2, bcryptjs 2.4.3                     |
